# About

This project shows off Remix's new MDX capabilities when using the [Vite plugin][remix-future-vite].
It also uses [Tailwind][tailwind] and the [typography plugin][tailwind-typography] for styling.

Inspired by [Rajesh Babu's MDX template][remix-mdx-blog] ❤️

[remix-future-vite]: https://remix.run/docs/en/main/future/vite
[tailwind]: https://tailwindcss.com/
[tailwind-typography]: https://tailwindcss.com/docs/typography-plugin
[remix-mdx-blog]: https://github.com/rajeshdavidbabu/remix-mdx-blog
